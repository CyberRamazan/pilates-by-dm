# MIND BODY pilates center — Landing prototype

Премиальный одностраничный прототип для студии Pilates в Актау (концепт для презентации команде).

## Stack

- Next.js 15 (App Router)
- Tailwind CSS
- TypeScript

## Запуск локально

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Деплой на Vercel

1. Загрузите репозиторий на GitHub.
2. Импортируйте проект в [Vercel](https://vercel.com).
3. Framework Preset: **Next.js** (настройки по умолчанию).
4. Deploy.

## Логотип

Положите файл логотипа в `public/logo-mind-body.png`.  
Если файла нет, отображается текстовый вариант «MIND BODY pilates center».

## Контакты (прототип)

Отредактируйте `src/lib/constants.ts`:

- `WHATSAPP_URL`
- `PHONE` / `PHONE_HREF`
- `INSTAGRAM_URL`
- `ADDRESS`
- `HOURS`
